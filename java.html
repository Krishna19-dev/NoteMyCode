Java Programming

Comprehensive notes on Java programming language

Introduction to Java

Java is a high-level, object-oriented programming language developed by James Gosling at Sun Microsystems and released in 1995. It is designed to be platform-independent through the concept of "Write Once, Run Anywhere" (WORA), thanks to the Java Virtual Machine (JVM), which allows Java code to run on any device that has the JVM installed.

Java is a compiled and strongly typed language, meaning code must be compiled before execution and variable types are strictly defined. It is widely used in enterprise applications, Android development, desktop GUIs, and backend systems.

Java's syntax is influenced by C/C++, but with a simpler object model and fewer low-level facilities. It supports features like automatic garbage collection, multithreading, and security.

Java is known for being robust, secure, and portable, making it a preferred choice for large-scale applications. It also supports object-oriented principles like encapsulation, inheritance, abstraction, and polymorphism.

Java Basics

Java syntax is clean and resembles C/C++ but is more object-oriented. A basic Java program must be written inside a class, and execution starts from the main() method.

// This is a comment in Java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!"); // Prints a string
    }
}

Variables and Data Types

String name = "John";      // String
int age = 30;              // Integer
double height = 5.9;       // Floating-point number
boolean isStudent = true;  // Boolean

Arrays

int[] numbers = {1, 2, 3, 4, 5};
String[] fruits = {"apple", "banana", "cherry"};

Conditional Statements

if (age > 18) {
    System.out.println("Adult");
} else if (age > 12) {
    System.out.println("Teenager");
} else {
    System.out.println("Child");
}

Loops

// For loop
for (String fruit : fruits) {
    System.out.println(fruit);
}

// While loop
int counter = 0;
while (counter < 5) {
    System.out.println(counter);
    counter++;
}

Methods (Functions)

public static String greet(String name) {
    return "Hello, " + name + "!";
}

public static void main(String[] args) {
    String message = greet("Java learner");
    System.out.println(message);
}

Java uses curly braces {} to define the scope of blocks, unlike Python's indentation-based structure.

Object-Oriented Programming in Java

Java is fully object-oriented (except for primitive types). Everything in Java revolves around classes and objects.

Defining a Class

public class Dog {
    // Class variable
    static String species = "Canis familiaris";

    // Instance variables
    String name;
    int age;

    // Constructor
    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Method
    public String description() {
        return name + " is " + age + " years old";
    }

    public String speak(String sound) {
        return name + " says " + sound;
    }
}

Creating Objects

public class Main {
    public static void main(String[] args) {
        Dog buddy = new Dog("Buddy", 9);
        Dog miles = new Dog("Miles", 4);

        System.out.println(buddy.description());
        System.out.println(miles.speak("Woof"));
    }
}

Inheritance

public class JackRussellTerrier extends Dog {
    public JackRussellTerrier(String name, int age) {
        super(name, age);
    }

    @Override
    public String speak(String sound) {
        return super.speak("Arf");
    }
}

public class Main {
    public static void main(String[] args) {
        JackRussellTerrier jack = new JackRussellTerrier("Jack", 3);
        System.out.println(jack.description());
        System.out.println(jack.speak("Woof"));
    }
}

Java supports all major OOP principles:

Encapsulation – Hiding internal data using private fields and providing access using getters/setters.

Inheritance – Extending a class using extends.

Polymorphism – Method overloading and overriding.

Abstraction – Using abstract classes or interfaces to hide complex implementation.

Java Libraries and Frameworks

Java’s ecosystem is vast and supports development in many domains:

Web Development

Spring Boot: A popular framework for building REST APIs and microservices.

Hibernate: An ORM framework for database interactions.

JavaServer Faces (JSF): A framework for building component-based user interfaces.

Mobile Development

Android SDK: The official development kit for Android applications.

Desktop Applications

JavaFX: A modern framework for building rich desktop applications.

Swing: An older framework for building GUI applications.

Game Development

LibGDX: A popular framework for game development in Java.

jMonkeyEngine: A game engine for 3D game development.

Data Science and Machine Learning

Weka: A collection of machine learning algorithms for data mining tasks.

Apache Spark: A powerful framework for big data processing and analytics.

Java-ML: A library for machine learning in Java.

Testing

JUnit: A widely used testing framework for Java applications.

Mockito: A framework for creating mock objects in unit tests.

TestNG: A testing framework inspired by JUnit but with more powerful features.

Java has a rich ecosystem of libraries and frameworks that extend its functionality for various applications, making it versatile for web development, mobile applications, data science, and more.

Java libraries and frameworks are essential for building robust applications efficiently. They provide pre-built functionalities, reducing development time and effort.

Some popular Java libraries and frameworks include:

  • Spring: A powerful framework for building enterprise applications.
  • Hibernate: An ORM framework for database interactions.
  • Apache Commons: A collection of reusable Java components.
  • JUnit: A widely used testing framework for unit testing.
  • Maven: A build automation tool for managing project dependencies.
  • Log4j: A logging utility for Java applications.

      • Java is a mature, stable, and high-performance language. Its strong community, massive library ecosystem, and platform independence make it ideal for everything from enterprise-grade applications to mobile apps, web development, and embedded systems.

      Whether you're building desktop tools or server-side systems, learning Java gives you the tools to build scalable, maintainable, and robust applications across platforms.
